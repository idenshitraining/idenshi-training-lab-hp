## ページ構成（セクション順）

1. **Header / Hero**
   - ロゴ＋ナビゲーション（固定ヘッダー）
   - キャッチコピー「調べて成果までお手伝い」
   - CTA ボタン「無料体験予約」

2. **Features（特徴）**
   - 遺伝子検査に基づくトレーニング設計などの強みを 3〜4 項目でカード表示

3. **Staff（スタッフ紹介）**
   - トレーナー 3 名分（ダミー名・写真アイコン・一言コメント）

4. **Price（料金プラン）**
   - 単発プランのみ（複数コースをカード形式で表示）

5. **Testimonials（お客様の声）**
   - 星評価付きレビュー 4〜6 件（ダミー）

6. **FAQ**
   - よくある質問を 5〜6 件アコーディオン形式で表示
   - チャットボットの応答辞書としても利用

7. **Access（アクセス）**
   - 住所・最寄り駅・営業時間・Google マップ埋め込み（ダミー枠）

8. **Footer**
   - コピーライト・問い合わせリンク

---

## チャットボット実装方針

- 右下固定の吹き出しボタンでチャット画面を開閉
- FAQ の Q&A をキーワードマッチで自動応答（JavaScript のみ、外部 API 不使用）
- 一致しない質問は「お電話・フォームよりお問い合わせください」と返す

---

## デザイン方針（CLAUDE.md 準拠）

- カラー：ホワイト × ネイビー（`#0a1f44` / `#ffffff` / アクセント `#1a73e8`）
- フォント：`'Noto Sans JP', sans-serif`（Google Fonts）
- レスポンシブ：CSS Grid / Flexbox、ブレークポイント 768px
- アニメーション：スクロール時に要素をフェードイン（Intersection Observer）

---

## 検証方法

1. ブラウザで `gym.html` を直接開き、全セクションの表示を確認
2. DevTools でモバイル表示（375px）を確認
3. チャットボットのボタンをクリックし、FAQ キーワードを入力して応答を確認
4. FAQ に登録していない質問を入力し、フォールバックメッセージを確認
